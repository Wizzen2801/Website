오토메이터 활용 - 네이버 사전 만들기

윈도우에서 batch 파일이 있다면 맥에서는 오토메이터가 있다.
허나 맥의 오토메이터가 더 활용도가 많다.

우선 간단하게 네이버 사전을 만들어 웹에서 모르는 영어 단어의 뜻을 찾도록 해 보겠다.
물론 모르는 단어를 네이버 사전 웹사이트에 접속해서 찾으면 되지만 오토메이터의 주된 활용은
여러번 클릭을 통해 수행해야 할 작업을 한번의 클릭으로 할 수 있게 해 준다.

런치패드안에 있는 오토메이터 실행하기 위해서는 4 손가락을 트랙패드에서 모은 후 아래의 오토메이터를 실행한다.



New document 클릭



Quick Action 선택



Run AppleScript 를 오른쪽 공간으로 드래그 해서 갖다 놓는다.



아래 스크립트 내용을 복사해서 붙여 넣는다.

on run {input, parameters}
return "http://endic.naver.com/popManager.nhn?m=search&query=" & (input as string)
end run
