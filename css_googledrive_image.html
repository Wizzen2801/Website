즉 아래와 같이 이미지가 깨져 보인다.



어떻게 하면 이미지를 보여줄 수 있을까 고민하던 차에 아래와 같은 웹사이트를 알게 되었다.
이곳에 링크를 복사하면 html 이미지 태그를 할 수 있게 생성 해 준다.
