웹사이트에서 유투브 자동재생 하기

유투브에서 웹사이트에 추가하고자 하는 영상을 선택 후



공유 버튼을 클릭하면 아래와 같은 화면이 보이는데 여기서 노란색 마크한 부분이 이 동영상의 id 값이다.



그럼 이 id를 아래 소스코드에서 웹사이트 주소의 embed/ 다음과 맨 마지막 부분인 "playlist=" 다음 값에 넣으면 된다.
여기서 autoplay 값을 1 을 설정한 것은 웹사이트 접속시 자동 재생을, mute 값을 0 은 음소거 하지 않도록,
그리고 loop 값을 1 을 설정한 것은 무한반복의 옵션 값이다.

"https://www.youtube.com/embed/ipCb7FyRll0?rel=0&autoplay=1&mute=0&loop=1;playlist=ipCb7FyRll0"


태그까지 포함한 완성된 소스코드는 아래와 같고 여기서 위에서 설명한 것과 같이 id 값을 변경 해 주면 된다.

1
<iframe width="100%" height="600px" src="https://www.youtube.com/embed/ipCb7FyRll0?amp;autoplay=1&amp;playlist=ipCb7FyRll0&amp;loop=1" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen=""></iframe>
cs

실행 해 보니 로컬에서는 잘된다.
그런데 웹에 올리면 자동재생은 되지 않고 반드시 클릭을 해 줘야 재생이 된다.
사파리, 크롬 모두 테스트 해 보았으나 동일하게 안되는 것을 보면 아무래도 해당 사이트를 접속시 갑작스럽게 음악이 재생되어 사용자들이 당황하게 되는 경우를 겪지 않도록 배려 한 것이 아닌가 하는 생각이 든다.
