버튼 클릭시 경고창 띄우기

이번에는 버튼을 클릭시 경고창을 띄우는 코딩을 해 보겠다.

위에서 보여지는 Alert 버튼을 클릭하면 경고창이 뜨면서 Alert 이라는 메세지가 보여진다.
2번째는 text 를 입력 받을 수 있는 input 태그인데 입력후 탭키 또는 엔터키등을 눌러 입력행위를 마치면 경고창이 뜨면서 changed 라는 메세지를 보여준다.
3번째는 onkeydown 이라는 명령을 쓰면 아무 키를 누르면 바로 경고창이 뜨게된다.

아래는 이에대한 소스코드이다. 


    <input type='button' value='Alert' onclick="
    alert('Warning!!!')
    "> 

    <input type="text" onchange="
    alert('changed!!!')
    ">

    <input type="text" onkeydown="
    alert('Key down!!!')
    ">
