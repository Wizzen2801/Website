CSS 란 Cascading Style Sheets 의 약자로서 html 이 웹사이트의 뼈대를 구성한다고 생각하면 CSS 는 html 에 옷을 입히는 역할을 한다고 할 수 있다.

일반적으로 CSS 는 head 태그 밑에 style 태그로 넣지만 보통 웹사이트를 구축 시 페이지 수 가 최소 10 페이지에서 최대 수백 페이지에 달하기에 CSS 파일을 별도로 작성해서 운영하는 것이 일반적이다.

예를들어 30번째 웹페이지를 작성했는데 첫번째 페이지 포함 모든 페이지의 속성을 변경하고자 할 때 CSS 파일을 별도로 운영하지 않으면 그동안 작성했던 모든 웹페이지의 style 태그 내용을 변경해야 하는데 이는 정말 생각하기도 싫은 작업이 될 것이다.

이에따라 CSS 의 내용을 "파일명.css" 란 확장자를 이용해서 저장해 놓고 html 의 head 태그에는 을 넣기만 하면 된다.

지금도 공부를 하고 있지만 CSS 란 녀석은 만만치 않은 놈이다.
특별히 조건을 지정하지 않았는데도 자꾸 충돌이 난다.
