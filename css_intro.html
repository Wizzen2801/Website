    Intro ...

    May 1, 2021

    html 이 공유하고자 하는 정보등의 내용을 보여주기 위한 도구라면 CSS 는 이를 잘 꾸며주는 도구이다.
    즉, 글꼴을 예쁘게 꾸민다던지 색을 입힌다던지 예쁜 장식의 노트안에 해당 내용을 넣는다든지 같은 것이다.

    예를들어,

    앞에서 입력한 "처음으로 웹페이지를 만들어 보았어요!" 라는 문구를 CSS 형식을 빌어서 아래와 같은
    소스 코드를 입력하여 꾸며보자.

    참고로 노트는 https://www.w3schools.com/w3css/w3css_notes.asp 에서 가져온 코드이다.

    우선 아래 링크를 <head> 태그안에 붙여 넣는다.

    1
    <link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">
    		cs

    그리고 아래 <div> 태그를 <body> 태그안에 붙여 넣는다.

    1
    2
    3
    <div class="w3-panel w3-pale-blue w3-leftbar w3-border-blue">
            <p>처음으로 웹페이지를 만들어 보았어요!</p>
    </div>
    Colored by Color Scripter
    		cs

    그럼 아래와 같이 예쁜 노트안에 문구가 나타나는 것을 볼 수 있다.

    기본적으로 CSS 는 <head> 태그와 <body> 태그 사이에 <style> 태그를 넣어서 그 사이에 CSS 코드를
    입력하는 것이 일반적이나 웹을 구성하는 형식이나 각종 태그들의 배치에 따라 해당 태그에 함께 포함하여 작성할 수 도 있다.

    CSS 소스코드들은 구글링하면 수 많은 정보들을 찾을 수 있기에 본인이 선호하는 코드들을 이용해서 웹사이트를 꾸며보자.
